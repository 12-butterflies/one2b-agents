## How to use Co-Work correctly

- **One thread per task.** Open a new thread for each distinct piece of work.
- **Archive when done.** Keeps the project clean and context sharp.
- **Never mix projects.** CEO Intelligence ≠ Studio Agent ≠ JASON OS.
- **Claude Code = build work.** Co-Work = operational and creative work.
- **These instructions are self-contained.** No local filesystem reads. Everything Claude needs is embedded inline.
