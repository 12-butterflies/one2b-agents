## While Cowork is down (until Tuesday 9 June)

Use Claude Code for everything. Open this app. Type your question. It handles all three session types.

For assets/video prompts, health questions, investor comms drafts — all here, same window.

---

## The rule: start fresh every day

Do NOT continue yesterday's thread. Start a new one. Paste the right context card. Ask your question.

Old thread = stale model = bad answers.
Fresh thread + context card = accurate model = good answers.

---

## How to start each session (30 seconds)

**BUILD session (Claude Code):**
Just open the app. CLAUDE.md loads automatically. You're ready.

**CEO session (Cowork):**
1. New chat
2. Paste `context-cards/CEO.md` (Cmd+A, Cmd+C, paste)
3. Ask your question

**JASON OS session (Cowork):**
1. New chat
2. Paste `context-cards/JASON-OS.md` (Cmd+A, Cmd+C, paste)
3. State your state (training/rest/tired/hangover) and ask

## For assets and videos

You do NOT need a separate session. Do it in BUILD (Claude Code).

Say: "Generate a LinkedIn post visual brief for One 2b, One2b voice, investor audience."
Or: "Write a Higgsfield video brief for a 30-second One 2b explainer."

Claude Code writes the brief. You execute it in Adobe CC / Higgsfield / wherever.
