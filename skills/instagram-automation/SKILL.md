# Instagram Automation

From ComposioHQ/awesome-claude-skills.

## What it does
Posts, stories, comments, media upload, and insights — via Composio Instagram integration.

## In One 2b context
- **Studio Social:** publishes Instagram content after Sentinel A1 + Jason approval
- **Scout:** retrieves post performance data for Fleet Learning weekly synthesis

**Never posts without Sentinel A1 PASS + Jason approval (onboarding gate).**

## Composio setup
Requires Composio API key + Instagram Business account OAuth.
Install: `pip install composio-core && composio add instagram`
